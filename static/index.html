Whisper + ChatGPT + gTTS

Fale com a EVA e receba a resposta em voz.

Grave sua pergunta no microfone ou envie um arquivo de áudio. O sistema transcreve com Whisper, gera a resposta com a API da OpenAI e sintetiza o áudio com gTTS para reproduzir direto no navegador.

Dicas rápidas

Teste frases como: "quero cancelar meu pedido" ou "preciso de ajuda com pagamento".
Se a chave da OpenAI não estiver configurada, o app usa uma resposta de demonstração para continuar funcionando.
O áudio gerado fica disponível pelo endpoint retornado na resposta da API.

Interface pensada para manter o visual amigável, mas agora alinhada ao fluxo de voz completo.
